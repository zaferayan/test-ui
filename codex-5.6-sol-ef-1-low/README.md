# React.js İnfografik Sayfası

Tasarım, verilen referans görselden bağımlılıksız HTML ve CSS ile uyarlanmıştır.

## Çalıştırma

`index.html` dosyasını tarayıcıda açın veya klasörde basit bir yerel sunucu başlatın:

```bash
python3 -m http.server 4173
```

Ardından `http://localhost:4173` adresini ziyaret edin.
